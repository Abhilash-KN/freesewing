# bruce

Bruce is a [freesewing](https://github.com/freesewing/freesewing) pattern
for a boxer briefs.

See [freesewing.org/patterns/bruce](https://freesewing.org/patterns/bruce)
for more info and examples.

## Install

```
npm i --save @freesewing/bruce
```
